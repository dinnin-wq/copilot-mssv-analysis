# Regularity Index Formula Implementation\n\n## Core Formula\n\n```python\nRe_g_vals = (E_vals**2) / S_vals\n```\n\n### Mathematical Definition\n\n```\nRe_g = E² / S\n\nWhere:\n  E    = Shannon Entropy (signal complexity/randomness)\n  S    = Mean Coherence (signal synchronization)\n  Re_g = Regularity Index (the output metric)\n```\n\n---\n\n## Component Breakdown\n\n### 1. Entropy (E)\n\n**Purpose:** Measures the complexity and randomness of the EEG signal\n\n**Calculation:**\n```python\n# Shannon Entropy from histogram\nE = -Σ(p_i * log(p_i))\n\nWhere:\n  p_i = normalized histogram bin values\n```\n\n**Implementation:**\n```python\ndef compute_entropy_values(data, n_bins=50):\n    \"\"\"Compute Shannon entropy for each channel\"\"\"\n    n_channels = data.shape[0]\n    E_vals = np.zeros(n_channels)\n    \n    for ch in range(n_channels):\n        # Create histogram of signal amplitudes\n        hist, _ = np.histogram(data[ch, :], bins=n_bins, density=True)\n        hist = hist + 1e-10  # Avoid log(0)\n        \n        # Compute Shannon entropy\n        E_vals[ch] = entropy(hist)  # scipy.stats.entropy\n    \n    return E_vals\n```\n\n**Interpretation:**\n- High E (2.0-4.0): Complex, chaotic signal (Wake/REM states)\n- Low E (0.5-2.0): Regular, predictable signal (Deep Sleep)\n\n---\n\n### 2. Connectivity (S)\n\n**Purpose:** Measures the synchronization between channels\n\n**Calculation:**\n```python\n# Mean coherence across all channel pairs\nS = (1/N) * Σ_pairs(Coherence(channel_i, channel_j))\n\nWhere:\n  N = number of channel pairs\n  Coherence = frequency-domain correlation (0 to 1)\n```\n\n**Implementation:**\n```python\ndef compute_connectivity_values(data, fs=128):\n    \"\"\"Compute mean coherence for each channel\"\"\"\n    n_channels = data.shape[0]\n    S_vals = np.zeros(n_channels)\n    \n    for i in range(n_channels):\n        coherences = []\n        \n        # Compute coherence with all other channels\n        for j in range(n_channels):\n            if i != j:\n                # Welch's method for coherence estimation\n                _, Cxy = coherence(\n                    data[i],      # Channel i\n                    data[j],      # Channel j\n                    fs=fs         # Sampling frequency\n                )\n                coherences.append(np.mean(Cxy))\n        \n        # Mean coherence with all other channels\n        S_vals[i] = np.mean(coherences) if coherences else 0.0\n    \n    return S_vals\n```\n\n**Interpretation:**\n- High S (0.6-1.0): Channels synchronized (Deep Sleep)\n- Low S (0.3-0.5): Channels independent (Wake/REM)\n\n---\n\n### 3. Regularity Index (Re_g)\n\n**Purpose:** Combines entropy and connectivity to classify sleep state\n\n**The Core Formula:**\n```python\nRe_g_vals = (E_vals**2) / S_vals\n```\n\n**Implementation:**\n```python\ndef compute_regularity_index(E_vals, S_vals, epsilon=1e-8):\n    \"\"\"\n    Compute regularity index: Re_g = E² / S\n    \n    Parameters:\n    -----------\n    E_vals : ndarray\n        Entropy values (n_channels,)\n    S_vals : ndarray\n        Connectivity values (n_channels,)\n    epsilon : float\n        Small value to avoid division by zero\n    \n    Returns:\n    --------\n    Re_g_vals : ndarray\n        Regularity index (n_channels,)\n    \"\"\"\n    # Numerator: Square the entropy\n    numerator = E_vals ** 2\n    \n    # Denominator: Connectivity with epsilon guard\n    denominator = S_vals + epsilon\n    \n    # Division: Create regularity index\n    Re_g_vals = numerator / denominator\n    \n    return Re_g_vals\n```\n\n---\n\n## Why This Formula?\n\n### Mathematical Rationale\n\n1. **Squaring Entropy (E²)**\n   - Amplifies differences in signal complexity\n   - Emphasizes high-entropy (chaotic) signals\n   - Non-linear relationship captures state transitions\n\n2. **Dividing by Connectivity (÷ S)**\n   - Normalizes by synchronization\n   - High S (sync) → Lower Re_g (organized)\n   - Low S (independent) → Higher Re_g (chaotic)\n\n3. **Combined Effect**\n   - Re_g measures \"chaos relative to synchronization\"\n   - Captures the fundamental difference between:\n     - **Chaotic but unsynchronized** (Wake/REM) → High Re_g\n     - **Organized and synchronized** (Deep Sleep) → Low Re_g\n\n---\n\n## Numerical Examples\n\n### Example 1: Wake/REM State\n\n```\nInput:\n  E = 2.8 (high entropy - complex signal)\n  S = 0.45 (low connectivity - desynchronized)\n\nCalculation:\n  Re_g = (2.8)² / 0.45\n  Re_g = 7.84 / 0.45\n  Re_g = 17.42\n\nResult: HIGH regularity index → WAKE/REM\n```\n\n### Example 2: Light Sleep\n\n```\nInput:\n  E = 2.2 (moderate entropy)\n  S = 0.52 (moderate connectivity)\n\nCalculation:\n  Re_g = (2.2)² / 0.52\n  Re_g = 4.84 / 0.52\n  Re_g = 9.31\n\nResult: MEDIUM regularity index → LIGHT SLEEP\n```\n\n### Example 3: Deep Sleep\n\n```\nInput:\n  E = 1.8 (low entropy - regular signal)\n  S = 0.68 (high connectivity - synchronized)\n\nCalculation:\n  Re_g = (1.8)² / 0.68\n  Re_g = 3.24 / 0.68\n  Re_g = 4.76\n\nResult: LOW regularity index → DEEP SLEEP\n```\n\n---\n\n## Sleep State Classification Thresholds\n\n```python\ndef classify_sleep_state(Re_g, E):\n    \"\"\"\n    Classify sleep state based on Re_g and E\n    \"\"\"\n    if Re_g > 2.0 and E > 2.5:\n        return \"Wake/REM\", 0.70  #   return \"Undetermined\", 0.50\n```\n\n**Threshold Interpretation:**\n\n```\nRe_g Value Range    Sleep State Moderate\nRe_g < 1.0         DEEP SLEEP (N3)         Simple + Synchronized\n```\n\n---\n\n## Implementation in Code\n\n### Complete Workflow\n\n```python\nfrom regularity_index_analysis import
